# 打卡小程序使用说明

## 📱 小程序简介

这是一款学习打卡小程序，支持英语、语文、数学等科目的学习打卡，通过 AI 智能识别和评分功能，帮助学生记录学习过程，提供即时反馈和学习建议。

---

## 🚀 主要功能

### 1. 首页打卡

**功能说明：**
- 选择科目（语文/数学/英语）
- 点击绿色圆形打卡按钮进行拍照打卡
- 支持查看历史打卡记录
- 支持 AI 智能评分

**使用步骤：**
1. 打开小程序，进入首页
2. 点击顶部的科目选择器，选择要打卡的科目
3. 点击中间的绿色圆形打卡按钮
4. 拍照或从相册选择图片
5. 等待 AI 识别和评分
6. 查看识别结果、得分和学习建议

---

### 2. 历史记录

**功能说明：**
- 按年月筛选查看历史打卡记录
- 按月分组显示，支持展开/折叠
- 查看打卡详情、评分、AI 分析结果

**使用步骤：**
1. 点击首页底部的"打卡记录本"进入历史记录页
2. 使用顶部筛选器选择日期和科目
3. 查看该时间段的打卡统计（累计打卡次数、得分、奖杯、小红花）
4. 点击月份标题栏可展开/折叠查看该月记录
5. 点击记录中的按钮可查看：
   - 🖼️ 打卡内容：查看原图
   - 📋 评价：查看检查结果详情
   - 💡 建议：查看学习建议

**列名说明：**
| 列名 | 说明 |
|------|------|
| 科目 | 打卡的科目（语文/数学/英语） |
| 日期/时间 | 打卡的具体日期和时间 |
| 得分 | AI 评分（0-10 分） |
| 内容 | 打卡内容预览，点击可查看原图 |
| 评价 | 检查结果，点击可查看详细信息 |
| 建议 | AI 学习建议，点击可查看 |

---

### 3. 工具箱

**功能说明：**
- 提供学习辅助工具
- 目前支持"扫描单词"功能

**使用步骤：**
1. 点击首页底部的"工具箱"按钮进入工具箱页面
2. 点击"扫描单词"卡片进入扫描页面

---

### 4. 扫描单词

**功能说明：**
- 拍照识别英语单词
- 自动语音朗读单词发音
- 显示单词音标、中文意思
- 提供常见例句和记忆技巧

**使用步骤：**
1. 从工具箱进入"扫描单词"页面
2. 点击拍照区域，拍照或选择图片
3. 等待 AI 识别单词
4. 识别完成后自动播放单词读音
5. 查看单词详情：
   - 单词拼写和音标
   - 中文意思
   - 常见例句（英文 + 中文）
   - 记忆技巧或口诀
6. 点击 🔊 按钮可重新播放读音
7. 点击"重新扫描"可识别新单词

---

## 📊 评分说明

AI 评分采用 10 分制：
- **8-10 分**（绿色）：非常棒！
- **6-7 分**（橙色）：还不错
- **0-5 分**（红色）：需要加油

**计分规则：**
- 同一天的多次打卡只计算最高分
- 每累计 100 分获得 1 个奖杯 🏆
- 每累计 10 分获得 1 朵小红花 🌸

## ⚙️ 使用提示

1. **拍照建议**
   - 保持光线充足
   - 尽量让文字清晰、完整
   - 避免手写字迹潦草

2. **网络要求**
   - AI 识别需要联网
   - 请确保网络连接稳定

3. **语音播放**
   - 扫描单词功能需要开启手机音量
   - 如无法播放，请检查手机静音设置

---

## ❓ 常见问题

**Q: 打卡后没有显示评分？**
A: 请检查网络连接，AI 评分需要联网处理。如仍无法显示，请重新尝试打卡。

**Q: 历史记录中没有显示某条记录？**
A: 请检查筛选器中的日期和科目设置，确保选择了正确的筛选条件。

**Q: 扫描单词识别不准确？**
A: 请确保图片中单词清晰可见，尽量拍摄完整的单词，避免模糊或遮挡。
