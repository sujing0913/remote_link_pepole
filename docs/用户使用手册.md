# 远程打卡小程序 - 用户使用手册

## 📱 小程序简介

远程打卡小程序是一款专为亲子互动设计的任务管理工具，帮助家长为孩子安排学习任务，记录孩子成长点滴。支持图片、视频等多种媒体形式，让亲子互动更加生动有趣。

---

## 👥 用户角色说明

小程序支持三种用户角色：

| 角色 | 说明 |
|------|------|
| **普通用户** | 未绑定任何关系的用户，可使用基础功能 |
| **家长** | 绑定了一个或多个孩子的用户，可为孩子安排任务 |
| **孩子** | 被家长绑定的用户，可查看家长安排的任务并打卡 |

---

## 🚀 快速入门

### 1. 首次使用

1. 打开小程序，自动完成登录注册
2. 进入「我的」页面查看当前身份
3. 根据需要绑定家长或孩子

### 2. 身份绑定

#### 成为家长
1. 进入「我的」页面
2. 点击「绑定监督人」
3. 生成绑定码，分享给孩子
4. 孩子输入绑定码后，建立亲子关系

#### 成为孩子
1. 进入「我的」页面
2. 点击「绑定监督人」
3. 输入家长提供的绑定码
4. 完成绑定，即可查看家长安排的任务

---

## 📖 功能详解

### 一、普通用户功能

普通用户是指未绑定任何亲子关系的用户，可使用以下功能：

#### 1. 首页
- 查看与自己相关的任务（近 7 天内）
- 浏览任务详情（内容、图片、视频）

#### 2. 任务卡
- 显示本周（周一至周日）的任务
- 顶部显示周次信息：「XXX 年第 X 周（X 月 X 日至 X 月 X 日）」

#### 3. 个人设置
- 查看个人信息
- 编辑昵称
- 绑定监督人

---

### 二、家长版功能

家长绑定孩子后，可使用以下专属功能：

#### 1. 首页 - 任务卡
- 查看所有绑定孩子的本周任务
- 通过孩子选择器切换查看不同孩子的任务
- 任务按创建时间倒序排列

**任务卡显示内容：**
- 📌 科目标签（语文📖、数学🔢、英语🔤、减肥🏃、生活🏠、健身💪、其他📌）
- 📝 任务内容
- 📅 创建时间
- ⏰ 截止日期（如有）
- 📷 附件（图片/视频）

#### 2. 安排任务

进入方式：
- 方式一：「我的」→「绑定孩子列表」→ 点击对应孩子的「📋 安排任务」按钮
- 方式二：首页底部导航 → 「安排」按钮

**安排任务步骤：**

1. **选择科目**
   - 点击科目选择器，选择任务类型
   - 可选：语文、数学、英语、减肥、生活、健身、其他

2. **选择年月**
   - 可切换年份（当前年和前一年）
   - 可切换月份（01-12 月）
   - 下方显示该年月的历史任务

3. **输入任务内容**
   - 在文本框中输入任务要求
   - 支持纯文字或文字 + 媒体文件

4. **上传媒体文件**（可选）
   - **拍摄/相册**：从手机相册选择或直接拍摄
   - **从聊天选择**：从微信聊天记录中选择文件
   
   > 💡 提示：如从聊天选择失败，请先将视频/图片保存到手机，再使用「拍摄/相册」功能

5. **设置截止日期**（可选）
   - 点击日期选择器，设置任务截止日

6. **提交任务**
   - 点击「创建任务」按钮
   - 成功后自动刷新历史任务列表

#### 3. 查看历史任务
- 在安排任务页面，选择年月查看历史任务
- 点击任务附件可预览
- 支持图片和视频预览

#### 4. 管理绑定关系

**查看绑定孩子列表：**
- 进入「我的」页面
- 滚动到「绑定孩子列表」

**批量解绑：**
1. 勾选要解绑的孩子
2. 点击「解绑已选」按钮
3. 确认解绑

**单个解绑：**
- 点击孩子右侧的「📋 安排任务」按钮旁边的解绑按钮（如有）

#### 5. 通知中心
- 孩子完成打卡后，家长收到通知
- 进入「我的」→「通知中心」查看详情
- 未读通知数量以红点显示

#### 6. 编辑昵称
1. 进入「我的」页面
2. 在「昵称」栏点击「编辑」
3. 输入新昵称
4. 点击「保存」

---

### 三、孩子版功能

孩子被家长绑定后，可使用以下功能：

#### 1. 首页 - 任务卡
- 查看家长安排的本周任务
- 任务范围：本周一到周日
- 顶部显示：「XXX 年第 X 周（X 月 X 日至 X 月 X 日）」

**任务卡内容：**
- 科目标签
- 任务内容
- 创建时间
- 截止日期
- 附件（图片/视频，可点击预览）

#### 2. 查看绑定家长
- 进入「我的」页面
- 在「我绑定的家长」列表查看

#### 3. 编辑昵称
- 同家长操作步骤

## 🔧 常见问题解答

### Q1: 从聊天选择视频失败怎么办？
**A:** 由于微信 API 限制，从聊天选择视频可能无法获取文件路径。解决方法：
1. 在聊天中长按视频
2. 选择「保存到手机」
3. 返回小程序，使用「拍摄/相册」功能选择

### Q2: 如何切换查看不同孩子的任务？
**A:** 家长在任务卡页面，点击孩子选择器（显示「👦 选择孩子：XXX」），选择要查看的孩子即可。

### Q3: 任务卡显示的时间范围是什么？
**A:** 任务卡显示本周（周一 00:00 至周日 23:59）创建的任务。

### Q4: 如何查看历史任务？
**A:** 
1. 进入「安排任务」页面
2. 选择年份和月份
3. 下方显示该年月的历史任务列表

### Q5: 解绑后还能看到之前的任务吗？
**A:** 解绑后，家长端不再显示该孩子的任务，孩子端也不再显示该家长安排的任务。历史数据仍保存在数据库中。

### Q6: 一个家长可以绑定多个孩子吗？
**A:** 可以。家长可以绑定多个孩子，并通过孩子选择器切换查看。

### Q7: 一个孩子可以绑定多个家长吗？
**A:** 可以。孩子可以绑定多个家长（如爸爸、妈妈），在「我绑定的家长」列表中显示。

### Q8: 如何收到打卡通知？
**A:** 
1. 确保已绑定孩子
2. 孩子完成打卡后，系统自动发送通知
3. 进入「我的」→「通知中心」查看详情

### Q9: 支持哪些类型的附件？
**A:** 
- 图片：jpg、jpeg、png、gif 等
- 视频：mp4、mov、avi 等（建议不超过 60 秒）

### Q10: 任务可以删除或修改吗？
**A:** 当前版本暂不支持删除或修改已创建的任务，请谨慎操作。
